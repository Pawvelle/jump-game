# 浮岛猫跃

一个轻量的网页版跳跃小游戏。玩家控制小猫在浮岛之间蓄力跳跃，连续精准落点可以获得更高连击分数。

## 玩法

按住屏幕蓄力，松开后角色跳向下一个平台。  
跳得越远分数越高，若未落在平台上则游戏结束。

## 操作

- 鼠标/触屏：按住蓄力，松开跳跃
- 空格键：按住蓄力，松开跳跃
- 左上角按钮：重新开始、暂停/继续、震动开关

## 特性

- Canvas 单文件实现，直接打开即可运行
- 分数越高，平台距离和精准落点要求会逐步提升
- 支持最高分本地保存
- 支持 PC 与移动端触控

## 运行

直接在浏览器中打开 `index.html` 即可运行，支持 PC 与移动端触控。
